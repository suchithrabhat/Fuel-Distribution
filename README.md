# Fuel Distribution Management System

## Overview
The Fuel Distribution Management System is a web application designed to simplify the process of fuel booking and distribution. It provides separate interfaces for customers, distributors, and administrators to efficiently manage bookings, fuel prices, payments, and feedback.

## Features
- User Registration and Login
- Fuel Booking
- Distributor Management
- Customer Management
- Fuel Price Management
- Payment Options (UPI and Card)
- Booking History
- Customer Feedback
- Responsive User Interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Flask
